Mini Data Center

Perangkat yang memiliki sistem terintegrasi yang dapat berfungsi sebagai data center mini. Terdiri dari satu atau lebih rak server yang sudah dilengkapi dengan sIstem pengkondisian udara, UPS, dan beberapa server. Server minimal terdiri dari 3 unit server dengan spesifikasi sama dan membentuk Hyperconverged Infrastructure (HCI). Teknologi HCI yang digunakan adalah teknologi terbuka yang memungkinkan integrase dan optimalisasi dengan beragam produk lain dengan teknologi yang sama. Di atas HCI tersebut, telah disediakan sebagai opto Sistem Cloud Storage dan Sistem Cloud Office di mana setiap user yang memanfaatkan system ini dapat berkolaborasi untuk document authoring, baik itu document word, spreadsheet, dan presentation

Command Center

  • Command Center merupakan pusat monitoring dan pengendalian kinerja serta pusat interaksi digital antara User dan Client.
  • Command Center ini terdiri dari sebuah ruang ‘Pintar’ (Smart Room).
  • Smart Room Command Center memiliki desain yang nyaman yang dapat memantau segala aktivitas sesuai fungsionalitas indikator kinerja selama 24 jam
  • Smart Room Command Center dilengkapi seperangkat Computer Workstation yang terhubung dengan TV Wall Display.
  • Terdapat beragam sistem aplikasi yang dapat menarik dan menampilkan informasi secara terpadu
  • Tersedia Mobile Quick Response Untuk Client
  • Tersedia Sistem Peta Digital yang dapat menampilkan informasi secara Spasial dan Historical.
Paket Command Center:
  1. Renovasi Ruangan.
  2. TV Wall.
  3. Workstation
  4. Wifi Hotspot
  5. Apache Kafka untuk Data Digestion dari berbagai sumber, seperti:
    • Aplikasi milik User
    • Social Media (Twitter, Facebook, News).
    • File-file laporan.
  6. Solr dan Hadoop sebagai Big Data Store.
  7. Aplikasi Mobile Quick Response.
  8. Aplikasi Dashboard Monitoring Quick Response.
  9. Aplikasi Cloud untuk Map dan GIS.
